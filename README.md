# JniSocketForAndroid
使用c++作为通讯核心  实现android socket长连接

Android socket communication using JNI Native development kit (NDK)

Both Server & client sides added

## Support
```
Android SDK upto 30
```
## Test Devices
```
-Samsung T235 Tablet Running Android 5.1
-Nokia 6.2 Running Android 10
```
